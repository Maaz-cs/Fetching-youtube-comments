# 📺 Fetching YouTube Comments using Python

## 📌 Project Description

This project is a simple Python program that fetches all comments from a YouTube video using the YouTube Data API.
It helps in collecting comments automatically instead of copying them manually.

This project is useful for:

* Data analysis
* Sentiment analysis
* Research projects
* Learning API usage

---

## 🛠 Requirements

Before running the project, make sure you have:

* Python installed (Python 3.x)
* Google API Client library

Install required library using:

```bash
pip install google-api-python-client
```

---

## 🔑 Setup Steps

1. Go to Google Cloud Console
2. Create a new project
3. Enable **YouTube Data API v3**
4. Generate an API Key
5. Replace the `API_KEY` in the code with your key

---

## ▶️ How to Run

1. Open the Python file
2. Replace:

   * `API_KEY` with your API key
   * `VIDEO_ID` with the YouTube video ID
3. Run the program:

```bash
python filename.py
```

The program will print all comments in the terminal.
